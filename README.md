# Clicker Counter

A desktop click counter application built with PyQt6 that tracks clicks and saves your high score across sessions.

## Features:
- Click counter that increases with each button press
- High score tracking
- Persistent storage - high score saved to file
- Reset button to start over
- Clean, simple interface

## How it works:
- Click the "Click Me!" button to increase your count
- Your high score is automatically saved when you beat it
- High score persists even after closing the app
- Use "Reset" to start counting from 0 again

## How to run:
1. Install PyQt6: `pip install PyQt6`
2. Run `python clicker-counter.py`
3. Start clicking!
